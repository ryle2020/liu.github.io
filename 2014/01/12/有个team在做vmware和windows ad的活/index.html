有个team在做vmware和windows ad的活

发表于 2014-01-12 分类于 程序人生 阅读次数： 本文字数： 838 阅读时长 ≈ 1 分钟

顺便去了解了一下vmware vSphere

一.什么是数据中心虚拟化
硬件设备转换成软件资源；虚拟化打破能耗循环；
物理硬件-》管理程序-》虚拟机
虚拟化的重要优势：合并 （合并在更少的硬件上，节约资金）
IT挑战：可用性，可伸缩性，最佳化，管理
二.vSphere的组件
1.虚拟机
2.EXSi管理程序：裸机管理程序，直接硬件访问，稳定性强，灵活的安装方式;托管方式管理程序-workstation
3.vCenter Server管理1000个ESXi主机，管理10000个虚拟机
vCenter组件：身份管理服务器，数据库服务器，应用程序服务器，web服务器，VMware vSphere Web Client
vCenter是具有管理功能的集中式平台
vMotion：将运行中的虚拟机不间断迁移
DRS：分布式资源调度：提供负载平衡，监控内存和cpu负载
DPM：分布式电源管理
StoragevMotion：迁移虚拟机的存储设备
存储DRS：存储负载均衡
vSphere Data Protection：备份虚拟机（VDP）还有开放的API：VADP
HA：高可用
FT：容错
vSphere Replication：将虚拟机复制到另外一个站点
4.存储和虚拟网络
存储：
vSphere支持：光纤，以太网光纤（FCoE）， iSCSI，NFS，Local
本地存储共享VSA vSphere Storage Appliance
vFlash：支持SSD
两种类型的数据存储：VMFS或者NFS
虚拟网络：
虚拟交换机：标准虚拟交换机， 分布式虚拟交换机
vSphere网络支持：VLANS，流量调整，端口镜像，服务质量QoS和DSCP，NetFlow，CDP和LLDP
5.vCenter Operation Manager，解决性能、容量和效率问题，监控动态：运行状况，风险，效率
6.vCenter Configuration Manager：更改发现和关联，自动修复，合规检查，配置分析
7.vCenter Site Recovery Manager：灾难恢复
