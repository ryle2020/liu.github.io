cubieboard用字符版的cubian，做了些配置，备注以免遗忘： 1.安装读exfat的package： apt-get install exfat-fuse 2.安装pycurl: apt-get install python-pycurl 3.安装pip: wget https://bootstrap.pypa.io/get-pip.py sudo python get-pip.py
