安装centos6.4虚拟机后的几个基本设置

发表于 2013-07-04 分类于 程序人生 本文字数： 378 阅读时长 ≈ 1 分钟

1.增加Sudoers

用su切换到root用户，cd /etc, chmod 777 sudoers, 然后vi sudoers

找到“root ALL=(ALL) ALL”这一行，下面增加”username ALL=(ALL) ALL”

最后chmod 440 sudoers

2.更改hostname

sudo vi /etc/sysconfig/network 然后修改hostname=需要设置的hostname

$hostname 需要设置的hostname

3.更新系统

sudo yum update

4.安装中文

sudo yum install ‘@Chinese support’

5.修改开放的端口

-M1 sudo setup （打开图形界面）

-M2 sudo vi /etc/sysconfig/iptables （默认开22端口，根据22端口的写法写想要开的端口）

service iptables restart

于是跟着小六，开始学点艾灸的东西。

《扁鹊心书》（未考证）“人于无病时，常灸关元、气海、命关、中脘......虽未得长生，亦可保百余年寿！“
1.艾叶：端午插门上，驱邪之物；
2.艾绒：青艾绒，陈艾绒，金艾绒，一般家里自用选陈艾绒-捏、观、闻、看其烟
3.艾条：看形、闻味、看火、观烟
4.艾粒：
5.艾炷：小的如同米粒-直接灸；大的犹如蚕豆-间接灸
艾灸方式：
  1. 隔姜隔蒜艾
  2. 隔盐艾
艾灸注意事项：
  1. 穿着宽松，除去一切可能阻碍气血运行的物品
  2. 取穴宜少，一般每次最多取3～4个穴位
  3. 时间宜长，一般单次20分钟左右
